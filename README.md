# NetSuite Customizations Portfolio

This repository contains a curated set of NetSuite customizations and
SuiteScript examples developed throughout my professional experience
as a NetSuite consultant.

## Purpose
The goal of this repository is to demonstrate:
- Clean and maintainable SuiteScript 2.x code
- Common NetSuite customization patterns
- Integration approaches
- Performance and governance-aware implementations

## Contents
- SuiteScript User Events, Client Scripts, Map/Reduce, and Scheduled scripts
- Generic RESTlet integration examples
- Reusable utility modules
- Architecture and best practices documentation

## Disclaimer
All code in this repository is **generic and sanitized**.
It does **not** contain client-specific logic, real account data,
or proprietary information.

## Technologies
- NetSuite SuiteScript 2.x / 2.1
- REST / JSON integrations
